Web App - Black and White to Colour - Using Django

Django, Python, OpenCv, ML, Computer vision

• As shown in the previous project (toColour), we know that a black and white image and video can be converted to a colored version.
• Machine learning projects like these have many real-world applications and can be very helpful for society, but their implementation is heavily dependent on hardware resources because machine learning requires a lot of GPU resources.
• What if we can upload these machine learning implementations on a cloud resource and provide easy access to it via a web app ?.
• Using the Django web framework, I implemented a Python-based web app, which allows users to access the black and white to color implementation remotely.

Year 2022

toColour - Black and White to Colour photo/video convertor

Python, OpenCv, ML, Computer vision

• toColour is an ML-based Python project.
• A DNN model is trained with various datasets, and the output of that training is saved as a Caffemodel.
• Using this model, we can convert an RGB image to a colored image.
• If we extend this idea even further, we can also convert a black and white video to a colored version.
• The video is a combination of many frames. From the input video, each frame is extracted sequentially, and that frame is colored and saved in an output location. Later, these colored frames are converted back into a video.
